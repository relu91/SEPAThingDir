# Delete Thing Descriptor

Delete the thing descriptor of **thingId** thing.

**URL** : `/thing/{thingId}`

**Method** : `DELETE`

**Auth required** : NO

**Data constraints**

--

## Success Response

**Code** : `200 OK`

**Content example**

```json
Thing added
```

## Errors

**Code** : `400 Bad Request`

**Content example**

```json
Thing id is missing
```
